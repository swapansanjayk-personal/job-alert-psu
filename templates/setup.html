\u26a0\ufe0f Gmail Users \u2014 IMPORTANT: Google blocks regular passwords for SMTP login. You MUST use a Gmail App Password.
  1. Go to your Google Account \u2192 Security
  2. Enable 2-Step Verification (if not already on)
  3. Search for "App Passwords" in Google Account settings
  4. Select "Mail" \u2192 "Other" \u2192 name it "PSU Job Alert"
  5. Copy the 16-character password and paste in the field above

For Outlook/Hotmail/Yahoo: Use your normal login password.

\U0001f680 Mailjet for Cloud Deployments: Render and other free cloud tiers block SMTP ports (25, 465, 587). Mailjet API sends email over HTTPS (port 443), which works everywhere.

Free tier: 200 emails/day — plenty for job alerts.

\U0001f680 Resend for Cloud Deployments: Simplest email API. Free tier: 100 emails/day. Works over HTTPS — no SMTP ports needed.

Sign up at resend.com → Add Sender → Verify email → Get API Key (starts with re_)

\U0001f4a1 Quick Tips
  • Enter the same email in both "Your Email Address" and "SMTP Username" for most providers
  • Set scan interval to 2 minutes for testing, then increase to 120 for regular use
  • The system automatically skips jobs you've already been notified about
  • You can manually trigger a scan anytime from the Dashboard
  • Use Mailjet or Resend when deploying to Render (SMTP ports blocked)
